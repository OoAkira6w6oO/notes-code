# 运行级别

| 级别 | 信息           | 说明         |
| ---- | -------------- | ------------ |
| 0    | 关机           |              |
| 1    | 单用户         | 找回丢失密码 |
| 2    | 多用户、无网络 |              |
| 3    | 多用户、有网路 | 最常用       |
| 4    | 保留级别       |              |
| 5    | 图形化界面     |              |
| 6    | 重启           |              |

# 修改系统开机时的默认级别

- /etc/inittab
  - = initialization table，table = 启动时的规则表

# 指定运行级别

- init [levelNumber]

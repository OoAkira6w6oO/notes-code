# 时间日期指令

- +号是、格式化输出选项
  - 有引号的话可以加入空格

| 指令                      | 功能                  |
| ------------------------- | --------------------- |
| date                      | 显示当前时间          |
| date + %Y/m/d/H/M/S       | 显示当前年/月日时分秒 |
| date "+%Y-%m-%d %H:%M:%S" | 显示当前年月日时分秒  |
| date -s [dateTimeString]  | 设置当前系统时间      |
| cal                       | 显示当前日历情况      |
| cal 2020                  | 显示2020年日历        |

# 基础思路

- 采用tree目录结构，只有一个根目录
- 在Linux中一切均为文件

# 指令文件

- /bin 常用指令
- /sbin 超级用户指令

# 日志文件

- /lost+found 非法关机后存放的某些文件
- /var
  - var其实是variable的缩写，存放动态可变文件
  - 日志 / 打印 / 邮件 / lock等

# 设备文件

- /dev 管理设备CPU、Disk等
- /media 媒体设备
- /etc 配置文件

# 系统文件

- /proc 内核相关
- /sys 系统
- /srv 服务

# 文件管理

- /mnt 挂载，让用户临时挂载别的文件系统
- /opt 安装文件
- /usr/local 通过编译源码的方式安装的程序
- /usr 用户安装的软件

# 其他

- /home 用户
- /lib 动态库
- /root 根用户
- /selinux 安全子系统
- /tmp 临时文件夹
- /boot 启动

# 网络监控netstat

- netstat = Network Statistics（网络统计）

- 监控网络服务状态
  - 显示系统的网络连接、路由表、网络接口状态等信息


### 常用指令

| 指令                      | 功能                         |
| ------------------------- | ---------------------------- |
| netstat -anp              | 查看系统所有的网络服务有哪些 |
| netstat -anp \| grep sshd | 查看sshd的网络服务           |

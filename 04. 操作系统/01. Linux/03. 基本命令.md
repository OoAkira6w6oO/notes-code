# 基础命令

| 命令            | 功能                                               |
| --------------- | -------------------------------------------------- |
| shutdown -h now | 立即关机                                           |
| shutdown -h 1   | 1分钟后关机                                        |
| shutdown -r now | 立即重启                                           |
| halt            | 关机                                               |
| reboot          | 重启                                               |
| sync            | 将内存数据同步到磁盘，关机和重启前都应使用sync命令 |
| man [cmd/ file] | 获得命令/配置文件的功能描述，获得帮助信息          |
| help [cmd]      | 获得shell内置命令的帮助信息                        |

## 循环语句

```python
# while循环
a = 0
while a < 10:
  a += 1
  if a == 5:
    break #跳出最近的循环体
    #continue 跳出一次循环
    
# for循环
for i in range(10):#本质：迭代器，0-9, 占用memory小 [0, 10)
  print(i)
for i in [1, 3, 5]:#列表、字符串、元祖集合均可
  print(i)
  
b = [1, 2, "st"]
for i in b:
  print(i)
for i in range(len(b)):
  print(i)
  
  
d = {'n':1, 'b':2} # 字典
for k, v in d.items():
  print(k)
  print(v)
```

# 入口

```python
if __name__ == "__main__":
  main()
  
"""
判断说明：
区分这个文件是“直接运行”还是“被当作模块导入”
如果这个文件是 直接运行的脚本，那么解释器会把 __name__ 设置为 "__main__"
如果这个文件是 被其他文件导入的模块，那么 __name__ 就会被设置成模块的名字（比如 "myscript"）

__name__ 是 Python 自动赋值的内置变量
直接运行时：__name__ == "__main__"
被导入时：__name__ == 模块名
"""
```

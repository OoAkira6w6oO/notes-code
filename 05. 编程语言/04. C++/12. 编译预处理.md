# 编译预处理命令

- 共同点
  - 都以`#`开头

# 宏定义

```C++
//常量宏定义
#define  ESC  0x1B

//带参数的宏定义
#define  AREA(r)  PI*r*r
```

# 文件包括

```C++
//自定义文件会被展开
#inclde "file1.cpp"

//头文件
#include <iostream>//编译器看到它就把对应文件内容插入到当前位置
```

- 每一个cpp文件，会被当作独立的文件编译成.o文件，不同的.o文件不会相互影响

# 条件编译

```C++
#include <iostream>

// 定义宏，用于模拟不同操作系统
// #define WINDOWS
#define LINUX

int main() {
#ifdef WINDOWS
    std::cout << "Running on Windows" << std::endl;
#elif defined(LINUX)
    std::cout << "Running on Linux" << std::endl;
#else
    std::cout << "Unknown OS" << std::endl;
#endif

    return 0;
}

```

# 其他

## line

- 指定该语句的下一行所在的行号
- 调试的时候非常好用

```C++
#line  1000
 int data=100;
 ……
//它表示int data=100;语句所在的行号为1000
```

## pragma

- 从源程序代码中向编译系统发送编译指令

```C++
// Person.h
#pragma once   // 告诉编译器：这个头文件只包含一次

#include <string>
class Person {
private:
    std::string name;
public:
    Person(const std::string& n);
    std::string getName() const;
};
```

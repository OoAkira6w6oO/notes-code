# 基础知识

- 异常处理 =  对错误情况的处理
- 在C语言中会有很多exit()

- 关键字
  - try
  - catch
  - throw

```C++
try {
          if (错误a) throw “Error 1”;
          if (错误b) throw 1;
          if (错误c) throw 2.34;
      }
catch (char *errstr)
{
     //具体错误处理
     exit(1);
}
catch(int errno)        //对应错误b
{
       //具体的错误处理
       exit(1);
}
catch (…)   //对应错误c
{
     //具体的错误处理
     exit(1);
}

```

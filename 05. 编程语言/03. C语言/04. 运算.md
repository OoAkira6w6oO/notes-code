# 增1、减1

- ++1，--1 是单目运算符（只有一个运算对象）
- 只能运算 int, char 的变量（常量不可）
- 前缀 ++n, --m
  - 先+后运算
- 后缀 n++, m--
  - 先运算后+
- 优先级顺序(由高到低)
  - -(负号), ++, --, *, /, %, +, -

# 赋值运算

- 赋值运算符是“＝”，它的作用是将赋值运算符右边的表达式

- 复合赋值运算符: +=，-=，*=，/=，%=

```C
x = x + 5;
x += 5;

x = x * (y + 1);
x *= y + 1;
```

# 运算符的结合性

- **左结合**：多个相同优先级的运算符时，从 **左往右** 分组

  - 例如：a - b - c
    实际是 (a - b) - c

- **右结合**：从 **右往左** 分组。

  - 例如：赋值 a = b = c
    实际是 a = (b = c)
  - 返回右值rvalue，右值不能被修改

- 前缀 ++i、--i → 单目运算符，右结合。

  后缀 i++、i-- → 单目运算符，左结合。

- 下面的表达式是错误的——因为rvalue不能被修改

  ```C
  ++i+++j;
  ++i++;
  (-i)++;//-i 是取反运算，结果也是一个右值
  ```

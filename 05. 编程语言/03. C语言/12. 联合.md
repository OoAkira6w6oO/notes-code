# 联合（共用体）

- 解决问题
  - 希望在不同时刻，将不同类型的数据放到同一个内存单元
  - 提供了在单个存储区域中操作不同类型数据的手段

# 基础知识

- 语法

  ```C
  union  联合名
  {
     成员表；
  } 变量表；
  ```

- 内存
  - 按照其占用存储区最大的成员来分配一份存储单元
- 引用成员
  - u.f
- 注意
  - 联合变量不能用作函数参数
  - 在定义联合变量时不能为其初始化

# 代码举例

```C
union un
{    
  float f;
  int i;
  char c;
} u;//分配一个float的空间

u.f=1.5;//必须指明成员
```
